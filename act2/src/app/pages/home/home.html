¿Qué es Angular?

Angular es un framework de desarrollo frontend mantenido por Google. Permite crear aplicaciones web modernas, rápidas y escalables mediante componentes, TypeScript y una arquitectura robusta.

Sobre este tutorial

Esta página web forma parte de una actividad de la asignatura Desarrollo de Aplicaciones en Red de la Universidad Internacional de La Rioja (UNIR).

El objetivo es crear un tutorial completo explicando una tecnología o framework, en este caso, Angular.
